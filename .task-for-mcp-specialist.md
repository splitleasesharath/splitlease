Task: Use mcp__supabase__execute_sql to find all email-related columns in the user table

SQL Query:
```sql
SELECT column_name, data_type 
FROM information_schema.columns 
WHERE table_schema = 'public' 
  AND table_name = 'user' 
  AND (column_name ILIKE '%email%' OR column_name ILIKE '%mail%')
ORDER BY column_name;
```

Goal: Return all email-related fields from the user table.
