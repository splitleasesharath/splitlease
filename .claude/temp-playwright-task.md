# Playwright Task: Capture List With Us V2 Page

## Objective
Navigate to http://localhost:8000/list-with-us-v2 and capture a full-page screenshot of the new SuperDesign component page.

## Steps
1. Launch browser (chromium)
2. Navigate to http://localhost:8000/list-with-us-v2
3. Wait for page to fully load (networkidle)
4. Take a full-page screenshot
5. Save screenshot to a temporary location
6. Close browser context

## Expected Output
A full-page screenshot showing the list-with-us-v2 page with the SuperDesign component rendered.
