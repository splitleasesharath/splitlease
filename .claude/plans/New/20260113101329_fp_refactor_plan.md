### Common Patterns

#### .push() -> Spread/Filter
```javascript
// Before
const arr = [];
if (condition1) arr.push(item1);
if (condition2) arr.push(item2);

// After
const arr = [
  condition1 && item1,
  condition2 && item2,
].filter(Boolean);
```

#### .sort() -> .toSorted()
```javascript
// Before
const sorted = [...arr].sort((a, b) => a - b);

// After
const sorted = arr.toSorted((a, b) => a - b);
```

#### for loop -> reduce/map
```javascript
// Before
const result = [];
for (const item of items) {
  if (item.active) result.push(item.name);
}

// After
const result = items
  .filter(item => item.active)
  .map(item => item.name);
```

// Before (in calculator/processor)
